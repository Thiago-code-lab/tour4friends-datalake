Data Lake
Medallion Architecture
Camada 01
Raw Zone
Bronze - JSON

Eventos originais extraidos do MongoDB via CDC, armazenados sem transformacao como fonte da verdade do pipeline.

Camada 02
Processed Zone
Silver - Parquet

Dados limpos, tipados e convertidos para formato colunar via AWS Glue, com schema validado e particionamento eficiente.

Camada 03
Curated Zone
Gold - Agregado

Tabelas agregadas com regras de negocio aplicadas, prontas para analise estrategica no Athena e consumo no Power BI.
